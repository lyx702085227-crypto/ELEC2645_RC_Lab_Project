# ELEC2645_RC_Lab_Project
##  Overview
This project goes beyond a simple calculator. It is a modular engineering tool designed to simulate, visualize, and analyze the transient response of RC circuits. It features a dashboard UI, fuzzy command parsing, and an AI-driven noise fitting laboratory.

##  Key Features
* ** Dashboard UI:** Real-time monitoring of R, C, and Tau parameters.
* ** AI Lab:** Features a "Noisy Data Fitting" module that uses Linear Regression to recover signal parameters from simulated noise.
* ** Smart Parsing:** Implements **Levenshtein Distance** algorithm to handle typos (e.g., understands `settup` as `setup`).
* ** Visualizer:** Custom coordinate mapping algorithm to render precise ASCII waveforms in the terminal.
* ** Quick Solver:** Bidirectional calculator for Voltage <-> Time conversions.

##  Tech Stack
* **Language:** C (C99 Standard)
* **Architecture:** Modular design (`.h` interface / `.c` implementation)
* **Dependencies:** Standard Libraries only (`<math.h>`, `<string.h>`, etc.)
